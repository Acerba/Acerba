Movelibs fix for VS2015 - MSB3073

Apparently a bug Visual Studio 2015 installation.

https://stackoverflow.com/questions/25566834/visual-studio-post-build-copy-command-error-msb3073-exit-code-1

Fix:

Double-click error message. Should open the file Microsoft.CppCommon.targets.
Right-click file name - Open containing folder -
(this should open something like 'C:\Program Files (x86)\MSBuild\Microsoft.Cpp\v4.0\V140')
right-click file - properties - toggle Read-only off -
open with some text-editing program as administrator - 
change :VCEnd</_BuildSuffix> to     </_BuildSuffix>
save file

recompile program
